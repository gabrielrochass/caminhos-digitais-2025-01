Bem-vindo à Loja Online!

Este site foi desenvolvido como um projeto de aprendizado, utilizando as tecnologias HTML, CSS e JavaScript.
A página apresenta uma navegação simples, um catálogo de produtos fictícios e um layout responsivo.
O design foi pensado para ser limpo e intuitivo, facilitando a navegação dos usuários.
Sinta-se à vontade para explorar e conhecer mais sobre como criar lojas virtuais com tecnologias web modernas!

Como foi feito:

  • Estruturação com HTML5
  • Estilização com CSS e Google Fonts
  • Funcionalidades básicas com JavaScript
  • Imagens ilustrativas e navegação simulada
